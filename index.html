Application Overview

StudyBuddy is a student productivity application designed to help users organize assignments, track deadlines, and study more effectively. From an end-user perspective, it provides a simple dashboard to manage school tasks and reduce stress.

Technology Overview

This application will be built using HTML, CSS, and JavaScript. It may also use frameworks such as React for the front end and Firebase for backend data storage. The project will be hosted using GitHub Pages.
